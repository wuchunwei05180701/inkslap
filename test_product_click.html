商品點擊功能測試

✅ 已實現的商品點擊功能：

  • 視覺反饋：商品卡片添加了 hover 效果和點擊動畫
  • 點擊處理：點擊商品會觸發 handleProductClick 函數
  • 購買動機記錄：顯示用戶對商品的興趣訊息
  • 商品詳細資訊：顯示商品的完整資訊
  • 聊天記錄整合：點擊後的訊息會添加到聊天歷史中

🔧 技術實現細節：

  • CSS 樣式：
    • 添加了 .clickable-product-card 類別
    • hover 效果：邊框變藍、陰影、向上移動
    • active 效果：輕微按下動畫
    • cursor: pointer 指示可點擊
  • 事件處理：
    • onClick={() => handleProductClick(product)}
    • 傳遞完整的商品物件
  • 訊息類型：
    • isProductInterest: 顯示興趣確認
    • isProductDetail: 顯示商品詳細資訊

📋 測試步驟：

  1. 在聊天組件中搜尋商品（例如：輸入「包」或「收納」）
  2. 等待商品推薦結果顯示
  3. 將滑鼠移到商品卡片上，觀察 hover 效果
  4. 點擊任一商品卡片
  5. 確認聊天記錄中出現：
    • 綠色背景的興趣確認訊息
    • 灰色背景的商品詳細資訊
